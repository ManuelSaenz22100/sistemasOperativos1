## Windows 10
#### Windows 10 es un sistema operativo desarrollado por Microsoft que se lanzó oficialmente el 29 de julio de 2015. Ha sido una de las versiones más recientes y populares de la serie de sistemas operativos Windows.

### Interfaz de Usuario: 
#### Windows 10 introdujo una interfaz de usuario moderna que combina elementos de las versiones anteriores de Windows, como el menú Inicio y la barra de tareas, con características nuevas y actualizadas. También se introdujo el tema "Fluent Design" para una apariencia más consistente y atractiva.

### Modelo de Servicio Continuo:
#### En lugar de lanzar una serie de nuevas versiones de Windows cada pocos años, Microsoft optó por un modelo de servicio continuo para Windows 10. Esto significa que Windows 10 recibe actualizaciones periódicas, incluyendo mejoras de seguridad, nuevas características y correcciones de errores.

### Cortana: 
#### Windows 10 incluye Cortana, un asistente virtual que permite a los usuarios realizar búsquedas, configurar recordatorios, controlar dispositivos inteligentes y obtener información útil mediante comandos de voz o texto.

### Microsoft Edge: 
#### Windows 10 presentó el navegador web Microsoft Edge, diseñado para ser más rápido y eficiente que Internet Explorer. A lo largo del tiempo, Edge ha evolucionado y ha ganado funcionalidades como extensiones y un rendimiento mejorado.
